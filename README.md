# AndroidAutomatic

AndroidAutomatic是一个可以在Android端实现自动化测试的一个辅助工具，无须冗余的复杂操作，加上可视化的脚本编写，让自动化回归简单，让小白也可以操作，目前处于1.0.0版，后续有时间会不断的优化操作。
